# JURIS-GESTIÓN-PRO ⚖️
### Sistema Integral de Gestión Legal y Protocolo Notarial

![Banner](assets/promo_hero.png)

**JURIS-GESTIÓN-PRO** es una suite de escritorio avanzada diseñada para profesionales del derecho y notarios en Honduras. Combina la potencia del análisis de datos con la inteligencia artificial para automatizar la carga operativa y garantizar la integridad jurídica en cada proceso.

---

## ✨ Características Principales

### 🧠 JURIS-AI (Asistente Inteligente)
- Autocompletado de demandas y escritos legales basados en plantillas institucionales.
- Extracción de datos del expediente para inyección directa en documentos Word.
- Consulta rápida de catálogos de incidentes procesales.

### 📜 Módulo de Protocolo Notarial (Fallo Cero)
- **Regla de Auditoría:** Validación estricta del límite de 200 folios por tomo.
- **Integridad ACID:** Garantiza la correlación cronológica y numérica de los instrumentos.
- **Exportación de Índices:** Generación automática de índices para auditorías de la Contraloría del Notariado.

### 🛡️ Escudo de Datos
- Sistema de respaldos automáticos `.jgpbackup` al cerrar la aplicación.
- Encriptación de base de datos local para máxima privacidad.
- Recuperación de PIN de administrador mediante código maestro de seguridad.

### 📊 Dashboard Analítico
- Visualización en tiempo real del estado de los casos (Activo, En Proceso, Cerrado).
- Calendario integrado de audiencias y actuaciones.
- Interfaz dinámica con soporte para Temas (Claro, Oscuro y Personalizado).

---

## 🚀 Instalación y Uso

1. **Descarga:** Obtén el instalador `Instalador_JURIS-GESTION-PRO_Full.exe` desde la sección de releases o la carpeta `installer`.
2. **Ejecución:** Sigue los pasos del asistente de instalación.
3. **Primer inicio:** La aplicación inicializará automáticamente la base de datos y los catálogos en tu carpeta local `%LOCALAPPDATA%` para garantizar la compatibilidad con Windows.

---

## 🛠️ Stack Tecnológico
- **Lenguaje:** Python 3.11
- **Interfaz:** HTML5 / CSS3 / JavaScript (Vanilla) vía `pywebview`
- **Base de Datos:** SQLite 3 (Capa de persistencia robusta)
- **Documentación:** `python-docx` para generación de documentos legales en formato oficial (Legal, Times New Roman 14pt).
